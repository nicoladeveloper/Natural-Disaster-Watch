📚 Documentação da API e Dados

Guia de consumo e referências sobre a fonte de dados utilizada.

Fonte de Dados

Os dados são fornecidos pela **NASA EONET** (Earth Observatory Natural Event Tracker), que rastreia fenômenos naturais globalmente.

Acessar EONET Original

Endpoint da API

GET http://127.0.0.1:8000/eventos

Retorna uma lista de eventos ativos formatada em JSON, incluindo título, coordenadas, categoria e link.

⚠️ Nota: Este endpoint é um *proxy* local para facilitar o desenvolvimento.

Estrutura da Resposta

Cada evento no array "eventos" possui:

  • titulo: Nome resumido do evento.
  • categoria: Categoria original da NASA (e.g., Wildfires).
  • coordenadas: Array [Longitude, Latitude].
  • data: Data de registro do evento.
  • link: URL para detalhes na fonte original.
